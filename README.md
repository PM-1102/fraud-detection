# Fraud Detection Project

A machine learning project for detecting fraudulent transactions.

## Project Structure

- **data/**: Raw and processed datasets
- **notebooks/**: Jupyter notebooks for EDA, preprocessing, modeling, and explainability
- **src/**: Source code for preprocessing, features, training, evaluation, and prediction
- **models/**: Saved model files
- **mlruns/**: MLflow experiment logs
- **app/**: Application files for serving predictions
- **tests/**: Unit tests
